## 工厂模式

主旨：将new操作单独封装，遇到new时就可以考虑是否需要用工厂模式

![工厂模式](https://github.com/cwzp990/summary/blob/master/images/工厂模式.png)

```

class Product {

  constructor(name) {
    this.name = name;
  }

  init() {
    alert('init')
  }

  fn1() {
    alert('fn1')
  }
}

class Creator {

  create(name) {
    return new Product()
  }
}

let creator = new Creator()
let p = creator.create('p')

p.init()
p.fn1()

```
